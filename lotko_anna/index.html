About

Greetings from the Internet (GftI) is a postcard making tool that allows users to craft a collaged graphic, include a custom message and send the postcard to a specified address. The process of image composition will enable users to select images from instagram or upload new images and customize SVG graphics with various widgets.

MVP Features

  • Search for Instagram images using keywords or upload custom images.
  • Select images from the search results as the body for masked letters.
  • Superimpose “Greetings from [search term]” text on the postcard graphics.
  • Customize background graphics with widgets that update the styles of SVG code.
  • Fill out a form that includes an address and a custom message.
  • Pay $1 to send the postcard using the Stripe API tools.
  • Confirm mailing via the LOB API.

Stretch Goals

  • Animations of example postcards being sent on the landing page.
  • Allow the user to preview the postcard and “flip” it to view both sides (3D animation).
  • Upload custom images.
  • Include map features that help users identify where postcards are going.

Project Notes

I am significantly updating the collage composition of the postcards from my Q1 project. In my Q1 project, I used a library that relies on HTML canvases and saved pixel data of the canvas in local storage. Instead I am going to rely on SVGs and create widgets that allow the user to update the vector graphics in real time. The LOB API allows users to send HTML code for postcard layout, including embedded SVG data. This will be my strategy for the composition.
