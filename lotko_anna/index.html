Greetings from the Internet

About

Greetings from the Internet (GFTI) is a postcard making tool that allows users to design a themed postcard and mail it. A user can manipulate SVG layouts with various widgets including image filters, boarders, fonts and colors. Users can also share their postcard designs with the GFTI community.

MVP Features

  • Upload an image and add filters to it.
  • Superimpose “Greetings from...” text on the postcard graphics.
  • Customize background graphics with widgets that update the styles of SVG code.
  • Fill out a form that includes an address and a custom message.
  • Pay $1.50 to send the postcard using the Stripe API tools.
  • Confirm mailing via the LOB API.

Stretch Goals

  • Animations of example postcards being sent on the landing page.
  • Allow the user to preview the postcard and “flip” it to view both sides (3D animation).
  • Upload custom images.
  • Include map features that help users identify where postcards are going.
