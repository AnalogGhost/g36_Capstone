Galvanize Mystery

A Three Part Computer Science Game by Keegan Ridgley

Answer a series of web development challenges to reveal clues

Answer questions about all three aspects of a web application to reveal final challenge

Final challenge gives the location of a hidden server

The hidden server hosts a "Chatroom of Champions" which is only accessible to users who have completed all CS challenge questions

FRONTEND

Frontend refers to the consumer facing visual and interactive side of web design. The frontend section of G | M introduces users to frontend concepts by engaging them with challenges that visually alter the Galvanize Myserty site as they progress through questions

MIDDLEWARE

Middleware connects websites to external data using asyncronous processes and JSON. In the Middleware section users replicate this by delivering data and traveling to real life locations to answer challenges and converting data from JSON

BACKEND

The Backend is a way of refering to data persistance and warehousing that happens behind the scenes in most websites. In the backend section users are introduced to REST and relational data structures through challenge questions that stress non-gui skills, object traversal and memory concepts
