BACKEND

The final question of Middleware leads users to the Location of a secret server hidden in the galvanize lobby. The users log on to the server and then solve a final series of challenge questions which are drawn from material that teaches the languages, concepts and metaphors of backend web development.

Users that manage to complete the final challenge are authorized to access a chat room hosted locally on the server

Challenge Subjects:

  • Command Line
  • Linux
  • PostgreSQL
  • Javascript
  • Servers
  • Data Manipulation
  • Convert hex / binary

Example Question

var X = 0x01000101100110110011011101111001
Convert var X into a decimal number
