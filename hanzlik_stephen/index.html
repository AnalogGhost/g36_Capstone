About My Project

Let It Grow is an open sourced browser based project that allows users to pull data from a indoor and outdoor gardens. An Arduino Uno, combined with a powerful weather sheild from Spark Fun and wifi capabilities by Electric Imp, sends http post requests to the server every second. Charts are used to vizualize temperature, humidity, light status, and soil moisture data in real time. This project exemplifies the power of the modern browser and it's ability to render a single page app powered by JSON data. Let it grow, let it grow, greatly yield!

Technologies

The technologies used include AngularJS, Javascript, NodeJS, ExpressJS, PostgreSQL, AJAX, Arduino, Bootstrap, Zing Charts
