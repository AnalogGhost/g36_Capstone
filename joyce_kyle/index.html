Visualize Revenue with React and D3

About this project:

The goal of this capstone project provide visual data analytics on order and delivery revenue for a local business. The application will consider data on orders totals and subtotals, deliver dates and times, and tips and provide both restauranters and delivery drivers with data-driven visualizations of their profitability.

Wireframe

User Stories: MVP

  • ○ The data visualization app is created in React components build with D3 that fit into the company's existing app
  • ○ Restaurant users should be able to access a professional invoice with their delivery totals for the month
  • ○ Restaurant users should be able view multiple charts populated with order and delivery information
  • ○ Restaurant users should be able to specify a date range to view within the month

User Stories: Stretch

  • ○ Driver users should be able to access a professional invoice with their delivery totals for the month
  • ○ Driver users should be able view multiple charts populated with delivery and tip information
  • ○ Driver users should be able to specify a date range to view within the month
