Capstone Proposal

I would like to create an app to promote deeper interpersonal connections through fostering open communication.

Functionality :

  • Users can access the original set of 36 questions created by Arthur Aron in his study "The Experimental Generation of Interpersonal Closeness" in Personality and Social Psychology Bulletin (1997)
  • Users can create their own decks of questions to run through with friends, family or anyone they feel the urge to create a stronger tie with.
  • Users can create personalized decks and style them with custom background images and fonts
  • Users can view, search, sort and play decks made by other site visiters.
  • Users have the ability to "heart" decks they want to play later.
  • Users can send decks to friends with shortened urls

Technologies :

  • HTML
  • CSS
  • postgreSQL
  • Express

APIs :

  • Rijksmuseum or MoMA
  • Google Fonts
  • Unsplash
  • Bitly

Unfamiliar Enviroment :

  • Angular2 or Vue.js
  • Foundation or Skeleton

MVP :

  • Wireframe
  • At least connect to either Rijksmuseum or Unsplash, and Google Fonts
  • Complete all user stories listed in functionality above

Stretch Goals :

  • Add Auth and User profiles
  • Word map of most used words in titles
  • Add location to decks and google map api (Show a more holistic community view)
  • Include some sort of three.js animation
