Northland Chorale

About

Since 1970, the Northland Chorale community show choir has been entertaining audiences with music and dance in a fun-filled family friendly show. A 501(c)3 nonprofit organization, the Northland Chorale is in need of a new website to showcase the organization and to attract audiences to its biannual shows.

MVP Features

  • Search for Instagram images using keywords or upload custom images.
  • Select images from the search results as the body for masked letters.
  • Superimpose “Greetings from [search term]” text on the postcard graphics.
  • Customize background graphics with widgets that update the styles of SVG code.
  • Fill out a form that includes an address and a custom message.
  • Pay $1 to send the postcard using the Stripe API tools.
  • Confirm mailing via the LOB API.

Stretch Goals

  • Animations of example postcards being sent on the landing page.
  • Allow the user to preview the postcard and “flip” it to view both sides (3D animation).
  • Upload custom images.
  • Include map features that help users identify where postcards are going.

Project Information

The Northland Chorale board of directors has asked me to create a new website for the choir as my capstone project. The project site will consist of a Home page, an About Us page, and a Show Information and Tickets page that includes functionality for audience members to purchase tickets to the April 2017 show. After completion of the capstone presentation, the site will be updated to include addition informational pages as well as a login section for members to have access to member-specific information and files.
