Description

Bag Tags is a site for disc golf enthusiasts to track and trade their bag tags. Bag tags are distributed at tournaments and are sold as a fundraiser. Each tag has a number on it, the lower the number the better ranking.
Bag Tag Tracking was created to inform and empower the disc golf community about their bag tags. This app is a way to track what tags you have and communicate with people who have the same bag tag as you so you can challenge them for their tag.

Features

Create a Profile
Upload an Image
View and Search all Tags
View Who has Which Tag
View Other Player's Profiles and Tags
Challenge Other Players via Email
View and Edit your Profile
Trade and Add Tags Effortlessly
