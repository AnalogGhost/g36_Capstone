Listed in order from first choice to last choice
#1: Life Planning
There are many processes and methods that have been created using the idea of Life Planning. This app will remove a lot of the confusion around going through this helpful and effective process. Users will be able to log in, evaluate their current circumstances, then plan goals based on that information, and refer to previously entered information. This information can be kept private or shared with others.
All of this data will be stored for easy accessability as needed at any point in time to stay up to date with planning. Users will be able to add reminders for action item due dates to Google Calendar. Find more information on wikiHow.
Technologies to be used: MEAN: MongoDB, Express, Angular, Node, Bootstrap, Google Calendar API
#2: Meeting Contact Tracker
This app will help users store information about recent contacts within their organization and sync that up with their meeting reminders for either Microsoft Outlook or Google Calendar. When a user meets someone new in their organization, they will be able to input personal, professional, or project related information about that person into their app.
Technologies to be used: MEAN: MongoDB, Express, Angular, Node, Bootstrap, Microsoft Outlook or Google Calendar API
#3: Language Speaker and
Learner Social Connections
This idea is intended to connect users who are learning a new language with users who already know that language. It should allow users to connect based on location as well as language.
Technologies to be used: MEAN: MongoDB, Express, Angular, Node, Bootstrap, Google Maps API
