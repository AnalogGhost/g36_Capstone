Shine

Description
  • Shine is a device that brings EMDR (Eye Movement Desensitization and Reprocessing) - a psychotherapy treatment for trauma - into the hands of all.

  • EMDR treatment helps access, process, & reformat traumatic memories bringing healing and allowing the user to finally SHINE.

  • Using external stimulus - such as directing the eyes with lateral movement - or in this case buzzers - the user focuses simultaneously on the traumatic event AND the buzzers.

  • Our incredible brain creates a steady ongoing response with the left-right movement as it travels through the trauma - such as one may train from one train station to the next.

  • People can remember the episode(s) but it no longer elicits distress. (woot!)
Features
  • Angular Material application connected to an arduino with 2 buzzers.

  • User will log in to access their treatment plan.

  • User will be able to enter length of time for treatment and number of sessions.

  • User holds the buzzers and the arduino produces left/right vibrations which simulate the traditional EMDR eye movement method.

  • Users can also seek out therapists using the Google Places API.

  • Stretch goals include badges for treatments completed, ability for users to rate their discomfort after treatments, to post messages in a community, and admin logins for therapists
Technologies

Arduino Uno with 2 LilyPad Buzzers, AngularJS, NodeJS, Express, PostgreSQL, Heroku

Zenout

Description
  • Zenout is a binaural beats generator to calm or stimulate brain waves.

  • Created with Javascript and the Web Audio API, this program pumps two different sound frequencies into separate ears.

  • The brain calculates the difference and starts resonating that calculated frequency difference - the binaural beat. This beat is correlated with a specific state of mind - from relaxed to peak concentration.

Technologies

HTML5, MaterializeCSS, jQuery, Javascript, Forismatic API, Web Audio API
