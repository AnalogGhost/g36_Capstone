Flora Find

About

Flora Find is a flower identification tool that uses Google's image recognition and Wikipedia to display information on a particular flower. It is designed for use on a mobile phone. A user can take a photo with their phone and upload it to retrieve information. They can also access an image from their photo library for later use.

MVP Features

  • Access the camera and photo library on a mobile phone.
  • Take a photo or upload to display the flower name.
  • Retrieve information about the particular flower.

Stretch Goals

  • Allow the user to save the photo with GPS coordinates of where it was taken.
  • Allow the user to share the image and location with others in the community.
  • Include functionality to identify plants as well.

Technologies

  • Phone Camera Hardware
  • HTML5
  • CSS3
  • Bootstrap
  • JavaScript
  • jQuery
  • Google Vision API
